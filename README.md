# Alien Invasion (外星人入侵)

一款依赖pygame和PyQt5的关于外星人的Python程序（书中小游戏改编）

## 作者信息

- **作者**：System32-MC | flb5 联合出品
- **版本**：v3.9.5 - beta.1
- **项目地址**：https://github.com/Chenbaojing123/alien_invasion

> [!TIP]
> 游戏包含多种模式，包括普通模式、道具模式、无限火力、Boss模式和娱乐模式，每种模式都有不同的游戏体验。

> [!IMPORTANT]
> 本游戏需要Python 3.8+、pygame和PyQt5支持，请确保在运行前安装所有依赖项。

> [!NOTE]
> 游戏包含成就系统，完成特定目标可以解锁成就并获得奖励分数。

## 游戏特色

- 经典的外星人入侵射击游戏
- 丰富的游戏系统和成就系统
- 超多的玩法
- 精美的游戏画面和音效

## 系统要求

- Python 3.8+
- pygame
- PyQt5

## 安装

```bash
pip install -r requirements.txt
python alien_invasion.py
```
