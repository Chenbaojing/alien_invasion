# 外星人入侵 (外星人入侵)

一款依赖pygame和PyQt5的关于外星人的Python程序（书中小游戏改编）

## 游戏特色

- 经典的外星人入侵射击游戏
- 丰富的游戏系统和成就系统
- 许多模式
- 精美的游戏画面和音效

## 系统要求

- Python 3.8+
-Pygame
- PyQt5

## 安装

```bash
pip install -r requirements.txt
python 外星人入侵.py
```
